# CSC311-Project

Final project of CSC311 Intro to Machine Learning. 

A [stacking ensemble](https://arxiv.org/pdf/0911.0460.pdf) of item response theory, matrix factorization, xgboost, and autoencoder models to predict the students' correctness of a given question using Eedi data. 

Placed 3/58 teams in the optional in-class [Kaggle competition](https://www.kaggle.com/c/csc311f20finalproject).
